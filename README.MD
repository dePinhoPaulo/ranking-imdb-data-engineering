# 🎬 Ranking IMDb - Data Engineering Project

## 📌 Overview

Este projeto tem como objetivo construir um pipeline completo de Engenharia de Dados utilizando o dataset **IMDb Top 1000 Movies Dataset**.

A solução contempla ingestão de dados em Lakehouse, transformação seguindo a Arquitetura Medallion (Bronze, Silver e Gold), modelagem da camada Gold para consumo analítico e desenvolvimento de dashboard interativo para geração de insights, além do versionamento de código com Git e GitHub, com separação de ambientes de desenvolvimento e produção.

---

## 🎯 Objetivos do Projeto

- Construir pipeline end-to-end
- Aplicar boas práticas de Engenharia de Dados
- Implementar arquitetura Medallion
- Dashboard com KPIs estratégicos
- Versionar e documentar o projeto
- Criar um case técnico para portfólio
